`@jiang-30/cli` 是 [whirls-admin](https://whirls.jiangbo.fun) 项目的脚手架和代码生成工具

## 安装

```bash
pnpm install --global @jiang-30/cli
```

## 命令

### 创建项目

```bash
whilrs init <project_name>
```

### 代码生成

```bash
whilrs generate
```
